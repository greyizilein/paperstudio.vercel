---
name: Writing Mode Definitions
description: Default and Natural modes both use canonical chapter headings; Natural mode varies voice only — never restructures outlines
type: feature
---
Writing modes define the engine's cognitive approach. **Both modes use the canonical dissertation heading schema as their structural backbone — neither mode renames, replaces, reorders, merges, or omits canonical headings.**

**Default mode**: AI writes with research writer awareness — context-aware, structurally guided. Consistent voice across generations.

**Natural mode**: Each generation produces writing that sounds like a different scholar wrote it — unique voice, rhetorical rhythm, argumentative temperament, intellectual signature per output. Natural mode's creative authority is **strictly over voice, argumentation, analytical depth, synthesis, and rhetorical strategy** — NOT over structure. Canonical main headings (e.g. "1.1 Background to the Study", "2.4 Empirical Review", "3.7 Method of Data Analysis") MUST appear verbatim.

Both modes MAY:
- Generate subheadings (e.g. 2.4.1, 2.4.2) within any canonical section
- Use study-specific naming for those subheadings

Voice Profile setting has been REMOVED — Natural mode inherently produces unique voices.
Language Level slider (1-7) remains and applies to both modes.
